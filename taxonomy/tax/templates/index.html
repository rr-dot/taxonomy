Taxonomy explorer

This is an application to explore the taxonomic classification (scientific names) of various species. Browse the taxonomical tree starting from the root node. The entry for each displays name, rank and if available a url to an image. Navigation links up are provided, as well as a breadcrumb menu for the full path from root node to current node.

The search feature has two fields, the first is the species name and the second optional field supplies the category (example: species). See for example this, this, etc.
